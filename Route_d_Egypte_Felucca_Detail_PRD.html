# Product Requirements Document (PRD): Felucca Tour Detail Page ## 1. Project Overview Create a high-fidelity React frontend for the "Felucca Tour" detail page, following the layout and functionality of the Route d'Égypte specific tour page. This page serves as a deep dive into a specific travel package, emphasizing itinerary, pricing, and booking. ## 2. Technical Stack - **Frontend:** React.js - **Routing:** React Router (for breadcrumbs and navigation) - **Styling:** Tailwind CSS (recommended for consistency with the design system) - **Components:** Modular React components for high reusability. ## 3. Visual Identity (Design System: L'Esprit d'Égypte) - **Primary Color:** Gold/Sand (#C79A3C) for accents, icons, and primary CTA. - **Backgrounds:** Clean white (#FFFFFF) with light gray (#F9FAFB) section alternates. - **Typography:** Plus Jakarta Sans for a modern, luxurious feel. - **Spacing:** Generous whitespace to maintain an "editorial" look. ## 4. Page Structure & Sections ### 4.1. Breadcrumbs & Header - Simple breadcrumb navigation (e.g., Accueil > Le Caire > Au départ du Caire...). - Integrated with the global `TopNavBar`. ### 4.2. Tour Hero & Gallery - Large H1 Title: "Au départ du Caire visites et croisière d’Assouan à Louxor en felouque". - Multi-image grid gallery: 1 large main image on the left, 4 smaller images on the right. - "Toutes les photos" button overlay on the bottom-right image. ### 4.3. Quick Info Bar - Horizontal grid showing: Durée (Duration), À propos (Tour Type), Taille du groupe (Group Size), and Langues (Languages). - Each item features a minimalist gold icon. ### 4.4. Content Layout (Two-Column) - **Left Column (Main Content):** - **Description:** Rich text intro about the circuit. - **Itinerary (Accordion):** Interactive vertical timeline/accordion for Day 1 to Day 10. Each day expands to show details. - **Inclusions/Exclusions:** Two distinct lists with checkmark (green) and cross (red) icons. - **Tarification:** Price table/summary section. - **Right Column (Sticky Booking Sidebar):** - A clean, modern contact/booking form with fields for Name, Email, Country, Phone, Nationality, and Message. - Primary CTA: "ENVOYER" (Gold background). ### 4.5. Footer - Global `Footer` component as defined in the design system. --- # Antigravity Design Prompt **Task:** Build a high-fidelity React frontend for the Tour Detail Page based on the visual reference and the L'Esprit d'Égypte design system. **Architecture:** - Use functional components and hooks. - Create a `TourSidebar` component for the sticky form. - Create an `Accordion` component for the itinerary. - Use `lucide-react` or similar for icons (Clock, MapPin, Users, Languages, Check, X). **Visual Style:** - **Primary Accent:** `#C79A3C`. - **Borders:** Subtle gray borders (`#E5E7EB`) for the sidebar and itinerary items. - **Shadows:** Soft `shadow-sm` on the sidebar. - **Typography:** Headlines in `Plus Jakarta Sans`, bold.
